## 🚀 Featured Projects

### 🔍 MemoryLens  
**Privacy-first AI-powered file search engine**

A local-first intelligent search system that lets users find files using **faces, text, and memory cues** — without uploading any data to the cloud.

- 🤖 Uses AWS Rekognition, Textract & Bedrock  
- 🔒 Fully local-first architecture (zero data exposure)  
- ⚡ High-speed indexing & search across large datasets  

---

### 🫁 Digital Twin for Lung Cancer Treatment  
**AI-powered healthcare simulation system**

A personalized 3D simulation of lungs and tumors generated from CT scans — enabling doctors to **test treatments virtually before real-world application**.

- 🏥 Simulates tumor response over time  
- 🧬 Predicts treatment side effects safely  
- 🧠 Bridges the gap between diagnosis and decision-making
